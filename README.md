# موقع د. إياس العكاري | Dr. Iyas Al Akkari Website 🏥

![Main Banner](picture/doctor-portrait-C-BgJ__U.jpg)

## 📝 وصف المشروع | Project Description
**بالعربية:**
موقع تعريفي واحترافي للدكتور إياس العكاري، أخصائي الطب التجميلي في دبي. يتميز الموقع بتصنيف الخدمات، عرض نتائج "قبل وبعد" بشكل تفاعلي، وقسم لآراء العملاء، مع تركيز عالٍ على تجربة المستخدم (UX) وسرعة الأداء.

**English:**
A professional portfolio website for Dr. Iyas Al Akkari, a leading Aesthetic Medicine Specialist in Dubai. The site features interactive "Before & After" comparisons, service categorization, and a testimonials section, all optimized for high performance and seamless UX.

---

## ✨ المميزات | Key Features
- **Responsive Design:** متوافق تماماً مع جميع الشاشات (Mobile, Tablet, Desktop).
- **Interactive Slider:** نظام مقارنة صور "قبل وبعد" تفاعلي باستخدام CSS & JS.
- **Scroll Reveal Animations:** تأثيرات ظهور العناصر أثناء التمرير (`reaval` class).
- **Dynamic Counters:** عدادات رقمية تظهر سنوات الخبرة وعدد العملاء بشكل حيوي.
- **WhatsApp Integration:** زر تواصل مباشر سريع عبر الواتساب.
- **SEO Optimized:** هيكلة HTML نظيفة لتحسين ظهور الموقع في محركات البحث.

---

## 🛠 التقنيات المستخدمة | Tech Stack
- **HTML5:** Semantic HTML for better SEO.
- **CSS3:** Custom properties (Variables), Flexbox, Animations.
- **JavaScript (Vanilla):** Scroll reveal logic, Mobile Menu, and Image Slider.
- **FontAwesome:** For professional medical & social icons.
- **Google Fonts:** Cairo Font for professional Arabic typography.
